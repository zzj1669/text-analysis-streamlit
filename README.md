# 文本词频分析Web应用
基于Streamlit开发的URL文本采集与多图表词频分析工具，支持7种可视化图表，自动过滤单字与停用词。

## 功能介绍
1. **URL文本采集**：输入文章URL自动爬取并清洗文本（过滤HTML标签、特殊符号）
2. **智能文本处理**：
   - 中文分词（jieba）
   - 强制过滤单字（长度<2的词汇）
   - 停用词过滤（支持自定义stopwords.txt，无文件时使用默认停用词）
3. **交互式词频筛选**：通过滑块过滤低频词汇（1-10可调）
4. **多图表可视化**（7种可选）：
   - 词云：直观展示高频词汇权重
   - 柱状图：横向对比词频差异
   - 折线图：展示词频变化趋势
   - 饼图：呈现词汇词频占比
   - 雷达图：多维度对比前6个高频词
   - 散点图：可视化词频分布
   - 漏斗图：展示词频层级关系
5. **数据展示**：表格呈现词频排名前20的词汇
